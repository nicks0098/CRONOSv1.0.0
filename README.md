# CRONOS AI

CRONOS AI is a premium local-first desktop AI assistant built with Electron, designed for private, fast, and flexible AI workflows.

## Features
- Local chat experience
- Model management
- Persistent settings
- Web Live mode with Docker-based SearXNG integration
requirements downloads
1. https://ollama.com/download/windows
2. https://www.docker.com/products/docker-desktop/ (only if you need the live web search)
## Windows Release
Download the installer from the Releases page.
steps
1. Install docker desktop website, https://www.docker.com/products/docker-desktop/
2. open the app and set for searXNG.
3. you will find the searXNG's host something like http://localhost:8888/
4. now go settings in the dicker desktop app, then general, then click "start Cocker when you sign in to your computer" and untick the option directly below it "open Docker Dashboard when Docker Desktop starts" and click apply.
5. go to the settings in the CRONOS AI app and paste http://localhost:8888/ in the searXNG host link, and click test.
6. if link is correct, then it would show success, then click save the changes.
7. now you can use the Web Live mode in CRONOS AI.
